Interference = selection by phase

Constructive when φ≈0, destructive when φ≈π. Quantum algorithms use this to amplify “good paths.”

RSA: easy multiply, hard to factor

This “one-wayness” is what Shor’s algorithm breaks by finding a hidden period.

    From r to factors

    Shor’s algorithm (concise)

    Input: odd composite N
repeat:
  1. Choose a ∈ {2,…,N−2}
  2. g ← gcd(a,N); if g > 1 → factor
  3. r ← QuantumPeriodFinding(f(x)=a^x mod N)
  4. if r odd or a^(r/2) ≡ −1 (mod N): continue
  5. return gcd(a^(r/2)±1, N)

    Quantum speedup sits inside the period-finding routine.
