What is computing?

Turning inputs into useful outputs via rules (algorithms).

Simple mental model: input → algorithm → output. Classical computers do this with bits and logic gates.

Where classical hits limits

  • Factoring very large integers (RSA)
  • Quantum chemistry / materials simulation
  • Combinatorial explosions

Nature is quantum… maybe our computers should be too.

Bit vs Qubit

Classical bit

Either 0 or 1

Qubit

Superposition: \( \,|\psi\rangle = \alpha|0\rangle + \beta|1\rangle,\; |\alpha|^2+|\beta|^2=1 \)

State lives on the Bloch sphere (we’ll use this visually, not algebraically).

Key ingredients

  • Superposition — explore many possibilities at once
  • Interference — reinforce right answers, cancel wrong ones
  • Entanglement — strong correlations across qubits

Quantum circuits

Sequences of gates that transform qubit states.

Think: programmable interference patterns.

Why be excited?

  • Exponential state spaces (powerful representations)
  • Speedups on specific problems (e.g., factoring, search, simulation)
  • New ways of modeling physical systems

RSA in 60 seconds

Security relies on: easy to multiply, hard to factor.

Public key uses N = p·q. If you can factor N, RSA breaks.

Shor’s big idea

Don’t factor directly. Find a period instead.

\( f(x)=a^x \bmod N \) is periodic with some smallest period \( r \).

Focus on r

  • Prepare a superposition over many \(x\)
  • Compute \( a^x \bmod N \) in a second register
  • Apply QFT → peaks at multiples of \(1/r\)
  • Measure → infer \( r \) (continued fractions)

From r to factors

If \( r \) is even and \( a^{r/2} \not\equiv -1 \pmod N \), then

\(\displaystyle \gcd\!\big(a^{r/2}-1,\,N\big)\ \text{ and }\ \gcd\!\big(a^{r/2}+1,\,N\big) \)

are non-trivial factors of \(N\).

Shor’s algorithm (concise)

Input: odd composite N
repeat:
  1. Pick a uniformly from {2,...,N−2}
  2. g ← gcd(a, N); if g > 1 return g
  3. r ← QuantumPeriodFinding(f(x)=a^x mod N)
  4. if r is odd or a^(r/2) ≡ −1 (mod N): continue
  5. return gcd(a^(r/2)−1, N) or gcd(a^(r/2)+1, N)

The quantum speedup sits inside the period-finding subroutine.

Reality check

  • Noisy qubits, decoherence
  • Scaling + error correction is hard
  • But progress is steady

Post-quantum crypto is being standardized.

Takeaways

  • Computers transform inputs to outputs; classical has limits
  • Quantum = superposition + interference + entanglement
  • Shor: factors via period r + QFT
