What is a computer?

Inputs → algorithm → outputs. In classical computing, rules act on bits with deterministic or probabilistic logic. Keep it visual.

RSA: easy multiply, hard to factor

This one-wayness is what Shor’s algorithm breaks by finding a hidden period.

Remind: public key uses N=pq. Security relies on factoring being hard classically.

Periodic function f(x)=a^x mod N

The period r of f is the secret structure. If r is even and a^(r/2)≠−1 mod N, gcd gives factors.

Quantum Fourier Transform (QFT)

QFT reveals sharp peaks at multiples of 1/r.

Key insight: interference concentrates probability at Fourier modes tied to r.

    Briefly say: rational approximation turns frequency info into an integer period.

    From r to factors

    If r is even and a^(r/2)≠−1 mod N, compute gcd(a^(r/2)±1, N).

    Shor’s algorithm (concise)

    Input: odd composite N
repeat:
  1. Choose a ∈ {2,…,N−2}
  2. g ← gcd(a,N); if g > 1 → factor
  3. r ← QuantumPeriodFinding(f(x)=a^x mod N)
  4. if r odd or a^(r/2) ≡ −1 (mod N): continue
  5. return gcd(a^(r/2)±1, N)

    Quantum speedup sits inside the period-finding routine.

    Land the full loop: classical pre-check, quantum r-finding, classical post-processing.
